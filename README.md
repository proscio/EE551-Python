# This is my new Markdown file
## Markdown is a Text-Editor:
Markdown allows for files to be written using certain modifiers in the text.
For example: 
<p>This is a paragraph in Markdown! It can contain anything such as <br>
- Line Breakes: </p> <br>
- **Bold Text** 
- *Italicized Text* 
- >block quotes 
- *and more!!!*
